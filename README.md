## 🚀 Project Overview

Professional website for the Department of Information Technology, TGPCET Nagpur featuring modern design with glassmorphism effects, smooth animations, and complete responsive layout.

## 🔧 Setup Instructions

1. **Download/Clone the project**
   ```bash
   git clone [repository-url]
   ```

2. **Add Images**
   - Place TGPCET logo in `images/college-logo/`
   - Add faculty photos in `images/faculty/`
   - Add lab images in `images/labs/`

3. **Add Documents**
   - Place IT Magazine PDF in `assets/documents/`

4. **Open in Browser**
   - Simply open `index.html` in any modern browser
   - No server required for basic functionality

## 🌐 Deployment

### Option 1: GitHub Pages
1. Push code to GitHub repository
2. Go to Settings > Pages
3. Select main branch
4. Your site will be live at `username.github.io/repo-name`

### Option 2: Netlify
1. Drag and drop project folder to Netlify
2. Site will be live instantly
3. Custom domain can be added

### Option 3: Traditional Hosting
1. Upload all files via FTP
2. Ensure proper file permissions
3. Point domain to hosting
